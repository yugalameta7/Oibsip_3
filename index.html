About TO DO

TO DO is an simple task management tool. Which is pureliy work on javascript. Below is the work flow of TO DO:
1. Fill the title box.
2. Fill up the description box.
3. Click on Add Task button.
4. Task will be added to pending task list.
5. Once the task is done user can mark it as complete by clicking on Mark Complete button.
6. After clicking on Mark Complete button, task is shifted to complete task list.
7. User can remove a task if required.
